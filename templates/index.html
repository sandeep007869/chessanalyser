UPLOAD PGN FILE

You can use any website to download PGN files, such as chess.com or lichess.org.

Things to keep in mind :

1. Make sure the PGN file contains the moves of the game and not just the metadata.
2. The file should have a section that lists the moves in standard algebraic notation.
3. The upload field must be filled and not be left empty.
4.There must be only single pgn file uploaded at once.
