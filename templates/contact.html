ABOUT THIS WEBSITE

This is a Chess Game Analyzer that evaluates moves from a PGN file using an engine. It classifies moves as best, good, mistake, or blunder to help users improve their gameplay.
PURPOSE
The goal of this project is to simplify chess analysis for beginners and provide clear insights into their games.
FEATURES
  • Upload or paste PGN files
  • Move-by-move analysis
  • Simple and clean interface
TECHNOLOGIES USED
  • Python (Flask)
  • HTML, CSS
FUTURE IMPROVEMENT
  • Add accuracy score
  • Improve UI and design
  • Add user accounts
